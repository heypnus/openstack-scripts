# 环境

## Aliyun

### Cli安装

```bash
pip install aliyuncli
# 账号登陆
pip install aliyun-python-sdk-aas
# 云服务器ECS
pip install aliyun-python-sdk-ecs

```

### 配置

需要填入账号的 access-id 和 access-secret
